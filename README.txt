BONJOUR,

Compilation du projet :

-Pour compiler notre projet, il suffit de faire la commande ./makeRobot.sh
-POur clean la compilation, il suffit de faire la commande ./cleanRobot.sh
-Prendre le .hex cree dans exec_libs pour faire la simulation

Avant de démarrer la simulation :

-Cliquer pour activer le voltage du bouton à pression
-Mettre du voltage pour chaque sonar et cliquer sur la case du bas pour les activer
-Ensuite, la simulation peut être démarée.
